# Telethon Auto Message Sender

Автоматическая отправка сообщений в Telegram каждый час.

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Создайте файл `.env` на основе `.env.example`:
```bash
cp .env.example .env
```

3. Получите API_ID и API_HASH:
   - Перейдите на https://my.telegram.org
   - Войдите в свой аккаунт
   - Перейдите в "API development tools"
   - Создайте новое приложение
   - Скопируйте `api_id` и `api_hash`

4. Заполните файл `.env`:
```env
API_ID=ваш_api_id
API_HASH=ваш_api_hash
PHONE=+7xxxxxxxxxx
RECIPIENT_USERNAME=username_получателя
MESSAGE_TEXT=Ваше сообщение
```

## Настройка

В файле `.env` укажите:
- `API_ID` - ID вашего приложения из my.telegram.org
- `API_HASH` - Hash вашего приложения из my.telegram.org
- `PHONE` - Ваш номер телефона (с кодом страны, например +79991234567)
- `RECIPIENT_USERNAME` - Username получателя (без @) или номер телефона
- `MESSAGE_TEXT` - Текст сообщения, которое будет отправляться

## Запуск

```bash
python main.py
```

При первом запуске вам будет предложено:
1. Ввести код подтверждения из Telegram
2. Если включена двухфакторная аутентификация - ввести пароль

После успешной авторизации программа будет отправлять сообщения каждый час.

## Остановка

Нажмите `Ctrl+C` для остановки программы.

## Примечания

- Файл `session.session` создается автоматически и хранит данные авторизации
- Не делитесь файлами `.env` и `*.session` с другими людьми
- Убедитесь, что у вас есть права на отправку сообщений получателю
